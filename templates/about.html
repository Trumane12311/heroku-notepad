Brief Project Summary & ETL

Finding Inspiration

Collectively as a team we elected to choose a topic that was fun, interesting and exciting to explore in more detail. We have enjoyed analyzing these types of data resources. Our ride during this project has had some ups, downs, bumps and head shaking experiences but overall the challenges that we encountered, have definitely strenghtened our ability to get to the finish line which is being demonstrated on the Heroku deployment link shown below.

Data Review & Analysis

We reviewed the various datasets from the RCDB (Roller Coaster Database) website which included data regarding Domestic & International Roller Coasters, Cencus Reports, Record Holders, etc. We realized that the data volume was quite high so we decided to write various scripts to web scrape the data.

Extract, Load & Transform

  • Developed various scripts to web scrape only the relevant data from the different sections of the website.
  • Loaded the web scraped data into various Excels and remote MongoDB for further analysis.
  • Consolidated the data that was collected into respective data frames while cleaning the data and renaming columns as needed.
  • Created interactive bar charts, pie plots, maps, etc.

Deployment

  • Via Heroku Cloud Application Platform.
